# World Happiness Report — Exploratory Data Analysis

## Overview
An exploratory analysis of the World Happiness Report dataset (2015–2019) 
across 150+ countries, investigating what factors drive national happiness 
and how scores have changed over time.

## Key Findings
- GDP, social support, and life expectancy are the strongest predictors of happiness
- Scandinavian nations consistently rank highest; conflict-affected states rank lowest
- Regional averages mask significant within-region inequality
- Global happiness scores remained stable from 2015–2019

## Tools Used
- Python, pandas, matplotlib, seaborn
- Jupyter Notebook

## Dataset
[World Happiness Report on Kaggle](https://www.kaggle.com/datasets/unsdsn/world-happiness)
